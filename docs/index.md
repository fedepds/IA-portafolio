# Portafolio — Inicio

Bienvenido al portafolio del curso. Aquí documentarás tu progreso, evidencias y reflexiones a lo largo del semestre.

## Curso y metodología
Este curso introduce a los estudiantes en el uso práctico de técnicas modernas de inteligencia artificial (IA) para resolver problemas reales en organizaciones. A través de unidades temáticas se trabajan Machine Learning, Deep Learning, Visión por Computador, Procesamiento de Lenguaje Natural y prácticas de MLOps. El enfoque prioriza el impacto organizacional: más que modelos sofisticados, se busca generar soluciones con valor claro. Se utilizan bibliotecas y modelos existentes, priorizando la capacidad de evaluar, adaptar e integrar soluciones. Las actividades prácticas están centradas en casos de uso con sentido estratégico, permitiendo que los estudiantes exploren problemas reales.

## Objetivos
- Aplicar técnicas modernas de IA para resolver problemas concretos de negocio.
- Comprender los fundamentos detrás de cada tecnología para poder evaluar sus posibilidades y limitaciones.
- Traducir necesidades organizacionales en soluciones técnicas con impacto.
- Utilizar herramientas y modelos de código abierto para implementar soluciones de manera ágil.
- Incorporar buenas prácticas de MLOps y DataOps para desplegar, versionar y mantener soluciones de IA en producción.

## Resultados de aprendizaje
- El estudiante será capaz de aplicar algoritmos de machine learning supervisado y no supervisado (regresión lineal, árboles de decisión, clustering) para resolver problemas de predicción y segmentación en contextos organizacionales, usando bibliotecas existentes.
- El estudiante será capaz de diseñar e implementar redes neuronales simples con frameworks actuales para abordar problemas que requieren Deep Learning (clasificación de imágenes, predicción en series temporales).
- El estudiante será capaz de construir soluciones con modelos de lenguaje y técnicas de NLP, incluyendo RAG, para tareas de análisis, extracción o asistencia conversacional, usando herramientas como Hugging Face, LangChain y Ollama.
- El estudiante será capaz de automatizar el ciclo de vida de un modelo mediante prácticas de MLOps, integrando componentes como versionado de datos, reentrenamiento y despliegue en APIs usando MLflow, FastAPI y Docker.
